# Mi Proyecto Django

## Descripción
Backend desarrollado con Django para el manejo en Expo de una app movil con Expo Go.

## Requisitos
- Python 3.8+
- Django 4.0+
- pip

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/jose-moncada/Backend-Movile.git
cd API SENA

# Crear entorno virtual
python -m venv venv
source venv/bin/activate  # En Windows: venv\Scripts\activate

# Instalar dependencias
pip install -r requirements.txt
```

## Uso

```bash
# Ejecutar servidor de desarrollo
python manage.py runserver 0.0.0.0:8000
```
